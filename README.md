# 应用授权管理系统

基于Flask的应用授权管理系统，支持临时和永久两种授权类型。

## 功能特性

### 授权类型
- **临时类型**: 生成应用码、密钥和指定的可用次数
- **永久类型**: 生成应用码和密钥，无使用次数限制

### 核心功能
- 用户登录和权限管理
- 创建和管理应用授权
- 验证应用授权
- 访问日志记录
- 应用状态管理（启用/禁用）

## 安装和运行

1. 安装依赖：
```bash
pip install -r requirements.txt
```

2. 运行应用：
```bash
python app.py
```

3. 访问系统：
- 管理界面: http://localhost:5000
- 默认账号: admin / admin123
